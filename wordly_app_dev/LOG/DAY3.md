## 1. 기본 규칙 숙지 (AGENTS.md)
> **"Read `AGENTS.md` first and follow it strictly."**

**💡 뜻:**
작업을 시작하기 전에 프로젝트 내에 있는 `AGENTS.md` 파일을 먼저 읽고, 그 안에 적힌 규칙을 반드시 지켜라.

**📝 `AGENTS.md`에 주로 포함되는 내용:**
- 프로젝트 구조
- 코딩 규칙
- 사용해야 하는 기술 스택
- 건드리면 안 되는 파일
- 실행 및 테스트 방법
- AI가 따라야 할 작업 방식

## 2. 온보딩 화면 완벽 구현
> **"Implement the onboarding screen as shown in the attached design exactly using assets from the assets folder."**

**💡 뜻:**
첨부된 디자인과 똑같이 온보딩 화면을 구현하고, 이미지나 로고 같은 자료는 `assets` 폴더 안에 있는 리소스를 사용해라.

**🔑 핵심 포인트:**
여기서 가장 중요한 단어는 **"exactly(정확히)"**입니다. AI에게 대충 만들라고 하는 것이 아니라, 아래의 요소들을 최대한 디자인과 동일하게 구현하라는 의미입니다.
- 레이아웃
- 색상
- 여백 (Margin / Padding)
- 폰트 크기
- 버튼 위치
- 이미지 위치
- 전체적인 화면 구성

---

## 3. 홈 화면 내 네비게이션 추가
> **"Add a navigation link on the home route to open the onboarding screen."**

**💡 뜻:**
홈 화면에 온보딩 화면으로 이동할 수 있는 링크나 버튼을 추가해라.

**📱 적용 예시:**
홈 화면에 아래와 같은 버튼을 생성하여, 클릭 시 온보딩 화면으로 넘어가게 설정합니다.
- `[ Go to Onboarding ]`
- `[ 온보딩 화면 보기 ]`
