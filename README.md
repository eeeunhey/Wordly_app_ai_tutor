# Wordly

**Wordly** is an AI-powered mobile language learning app that helps users build vocabulary and speaking habits through interactive lessons and real-time AI voice tutoring.

워들리(Wordly)는 사용자가 단어와 말하기 습관을 자연스럽게 쌓을 수 있도록 돕는 AI 기반 모바일 언어 학습 앱입니다.  
게임형 학습 흐름과 실시간 AI 음성 튜터 기능을 통해, 사용자가 부담 없이 매일 언어를 연습할 수 있도록 설계했습니다.

## 📱 About the Project

Wordly is a mobile-first language learning app designed to help users build consistent vocabulary and speaking habits through short, interactive lessons and real-time AI voice tutoring.

Unlike traditional lesson-based learning apps, Wordly focuses on creating a more responsive practice experience where users can speak, receive instant feedback, and continue learning through natural AI-powered interaction.

워들리는 사용자가 짧고 인터랙티브한 학습을 통해 단어와 말하기 습관을 꾸준히 쌓을 수 있도록 설계된 모바일 중심 언어 학습 앱입니다.

기존의 정해진 문제 풀이형 학습을 넘어, 실시간 AI 음성 튜터와 직접 말하고 반응을 주고받으며 더 자연스럽게 언어를 연습하는 경험을 목표로 합니다.

## 🎯 Core Concept

> Learn words. Practice speaking. Build a daily language habit.

사용자는 짧은 학습 단계를 통해 단어를 익히고, 실시간 AI 튜터와 대화하며 말하기 감각을 키울 수 있습니다.

---

## 🚀 Main Goals

- Make language learning feel simple and playful
- Help users build consistent daily learning habits
- Provide real-time AI voice interaction
- Create a clean and modern mobile UI
- Manage authentication, user state, and analytics in a scalable way
